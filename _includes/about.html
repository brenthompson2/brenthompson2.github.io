Software Development

Up until recently, just about all of my legitimate coding experience had been from creating console applications in object-oriented paradigms. This includes simulating an Operating System in C, designing a compiler in C++, and countless other applications for the Transylvania University Computer Science program. Last May I handled all of the coding for a video game, Traffic Cop Hero 1000, which was developed using the Unity engine and scripts written in C#. This summer I had an absolute blast working on my own project, which was designing Audio Software Synthesizers using C++ and the JUCE Audio Framework. I have also worked on projects in Java, Common LISP, Microsoft Visual Basic, Prolog, and MIPS Assembly Language. The only mobile application development that I have done was using the hybrid mobile app framework Ionic 3, which utilizes the power of web technology to integrate native code and maintain a smooth, responsive UI.

    Throughout the years I have dabbled with web technology, but it wasn't until starting as a Web Development Intern at Awesome Inc that I really got into the nuts and bolts. I quickly had to teach myself the ins and outs of concepts like asynchronous data flow; countless layers of abstraction; a hodgepodge of interconnected languages such as HTML, CSS, Sass, JavaScript, TypeScript, Markdown, and PHP; and an absurd amount of frameworks and technologies such as Angular, Ionic, Bootstrap, and Jekyll. I started out managing a few of Awesome Inc's many internal websites but in no time had proven an aptitude with the technology and thus began working on actual customer websites. Most of the client websites were managed using WordPress, but there were a few custom child themes. I also was the creator of their Check-In application that all of their guests now use upon entry to the facility. This mobile application was designed using the hybrid mobile app framework Ionic 3, which leverages the power of advanced technologies including Cordova, Angular, Bootstrap, and many more.
