    Formant Analysis
  • Created Patches that analyze, synthesize, and filter Formants
  • Developed using advanced Max MSP software

    • There are two different patches within this project. The Formant Generator is a sine wave synthesizer that produces vowel sounds whereas the Formant Filter takes advantage of the Fast Fourier Transform in order to silence all frequencies below the selected cutoff amplitude. They was developed for the fall 2017 Music Cognition class at Transylvania University
